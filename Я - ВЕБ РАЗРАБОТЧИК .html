Структуры данных и алгоритмы

Важные предметы для повышения уровня и Для собеса. [ 2 месяца ]

[ БАЗА ДАННЫХ ] Проектирование Баз Данных

Существует два типа базы данных

Реляционные

[ MySql — PostgreSQL — SQL Server — Oracle ]

Во всех этих используется язык — SQL

  • Данные хранятся в таблицах со строками и столбцами которые связаны между собой.
  • Лучше всего подходит для приложений требующих сложных запросов

Нереляционные ( NoSql )

[ MongoDB — CouchDB — Cassandra ]

  • Данные хранятся без определенной структуры.
  • Лучше всего подходит для приложений требующих гибких моделей данных

Веб Фреймворки [ 2 месяца ]

  • Зависит от выбранного вами языка программирования.
    • JavaScript —> Express.JS
    • Python —> Django
    • PHP —> Laravel
    • Java —> Spring Boot
    • C# —> .NET Core
    • Ruby —> Ruby on Rails
    • GO —> Gin

Все эти фреймворки выполняют одну задачу

Предоставляют набор инструментов для создания Back-End

  • Маршрутизация
  • Обработка запросов / ответов
  • Работа с базами данных
  • Обеспечение безопасности

С помощью этих фреймворков мы можем создавать API (Интерфейсы программирование для взаимодействия)

API — это инструмент связи между фронтэндом и бэкендом. Позволяющий отправлять или получать данные между собой.
